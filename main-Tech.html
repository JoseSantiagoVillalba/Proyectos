Variables en Javascript

Las variables son conceptos significativos en programación. Que todo estudiante debe aprender. JavaScript, en particular, puede declarar variables usando las palabras clave var, const o let.

En la programación utilizamos datos como información en nuestros programas informáticos. El número de ciertas frutas que compré en una verdulería podría considerarse un tipo de dato. Así como el nombre del vendedor/a, el precio del producto, el tipo de billete con el que pagamos y el vuelto que se nos da.

De esta manera podemos ver:

  • var frutas= 1;
  • var nombreVendedor= “Marta”
  • var precioDelProducto= 20
  • var billete= 50
  • var vuelto= 30

Como vemos los datos en la programación son muy importantes. Para manipular datos, los programadores necesitan alguna manera de guardar y registrarlos. Es aquí, donde entran las variables.

Una variable es el nombre de un contenedor en la memoria que contiene un tipo de información, por ejemplo 1, Marta, 20, 50 y 30. Imaginemos a una variable como una caja a la que debemos acceder para encontrar cierto tipo de información(dato).

Variable Var

Las variables que se escriben con var para definir cualquier tipo de variable que se puede modificar.Además permite crear variables con alcance a su función más cercana.Tiene por defecto el valor undefined. Lo que quiere decir puede no estar definida.

Por ejemplo un perro no tiene nombre y luego viene una persona y se le da uno, entonces la variable nombre utilizará esta nomenclatura.

Variable let

Las varibles escritas con let necesitan un valor por defecto. No pueden ser llamadas en valor indefinido como var. También permite definir variables con un alcance a su bloque más cercano.

Variable const

Las variables que se escriben con const para definir cualquier tipo de variable que es inalterable o constante.

Por ejemplo el nombre de una persona, que siempre será el mismo.

Concusión

Por lo tanto, si necesitamos que nuestro programa tenga variables indefinidas que luego definirán usaremos var. Si, por el contrario necesitamos que todas las variables estén definidas por un valor utilizaremos let. Estas variables se pueden cambiar.

Por otro lado, si queremos que las variables no puedan ser cambiadas en ningún momento, usamos const.
